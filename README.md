# chatGBT
使用自然语言生成行为树的引擎

# 安装需要
## 1 py_trees 相关
- pydot

## 2 asr 相关
- 使用的是paddlepaddle的框架
```pip
    conda install paddlepaddle-gpu==2.4.1 cudatoolkit=11.7 -c https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/Paddle/ -c conda-forge
``` 
- 其他依赖包，注意需要安装C++14以上
```pip
    pip install requirements.txt
```
